# Real-Time Object Detection

This project contains a Python script for real-time object detection using OpenCV and the YOLOv3 model.

## Prerequisites

- Python 3.x
- OpenCV (`cv2`)
- NumPy

## Model Files

To run this project, you will need the YOLOv3 weights file (`yolov3.weights`). Ensure that you have the following files in the same directory as the script:
- `yolov3.weights`
- `yolov3.cfg`
- `coco.names`

## Usage

Run the main script:
python main.py

It will open your webcam and start detecting objects in real-time. For exiting, press the 'q' key.
